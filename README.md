# Dellini AI - AI Readiness Scorecard

A modern, interactive assessment tool for manufacturing companies to evaluate their AI operations readiness.

## Features

- 12-question interactive scorecard
- Real-time scoring and feedback
- Email delivery of personalized reports
- Responsive design with Dellini branding
- Fast and lightweight

## Setup

### Install Dependencies

```bash
npm install
```

### Environment Variables

Create a `.env` file with:

```
EMAIL_USER=your-email@gmail.com
EMAIL_PASSWORD=your-app-password
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Production

```bash
npm start
```

## Deployment

This project is optimized for Vercel deployment.

1. Push to GitHub
2. Connect to Vercel
3. Add environment variables in Vercel settings
4. Deploy!
